Create a Room

Create a room with a custom code, set a specific time duration, define the maximum number of guests allowed, restrict access if needed, and extend the duration later.

Join a Room

Enter the room code to join. If the room has no restrictions, you'll join instantly. However, if it's a restricted room, you'll need to wait for the host's approval.

Instant Sharing

Create an instant room with a fixed code. The guest count cannot be set initially, but you can restrict access, increase the guest limit, and extend the duration later.
